# 使用 vscode 断点调试 Rust 程序

## 步骤
* 打开 vscode，在插件市场搜索 https://marketplace.visualstudio.com/items?itemName=vadimcn.vscode-lldb，点击 install 即可
* 安装过程中，会下载一个包（Mac 机器上），需要等待一段时间，下载完成后，重启 vscode
* 至此，你可以对你的 Rust 程序进行调试了
